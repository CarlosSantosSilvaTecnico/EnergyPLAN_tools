#EnergyPLAN TOOLS

This repository includes add-ons that you can donwload and place in the EnergyPLAN Tools folder.

At the moment it includes the following tools:

- EPLANOptGUI.exe -  is an autonomous executable application that allows EnergyPLAN users to use solve the capacity expansion optimization problem of energy systems
through a graphical user interface. The tool is programmed in Python and is based on the EPLANopt library (2023), developed by Matteo Prina from Eurac Research
[EPLANopt](https://github.com/matpri/EPLANopt)
